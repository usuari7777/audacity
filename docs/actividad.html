Actividad

Eliminación del ruido de fondo

Temporalización: 20 minutos

Archivos necesarios: (hacer clic con el botón derecho del ratón sobre el enlace y guarda el enlace en el disco para poder trabajar con el archivo)

  • Dos niñas hablando.mp3

 

Utiliza el archivo sonoro dos niñas hablando.ogg en el que se ha grabado una conversación entre dos niñas pero en el que hay el ruido provocado por una màquina muy molesto. Intenta eliminar el ruido de fondo de forma que el resultado final quede:


"Juega" con el nivel de eliminación de ruido para que no se distorsione mucho la voz de las niñas
