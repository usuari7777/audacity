2.2.1. Extracción de las pistas de un CD audio

Extracción de audio digital a partir de un CD audio

Temporalización: 30 minutos

Si has intentado copiar las canciones de un CD audio para hacer una copia de seguridad de las canciones contenidas en él o pasarlas al reproductor MP3, habrás comprobado que después no es posible reproducirlas, para poder reproducirlas primero las has de pasar a otro formato reconocible por el ordenador o estos dispositivos como el formato OGG, MP3, WAV, ...

Existen muchas aplicaciones que realizan esta función. Lliurex lleva instalado el programa Extractor de CD Sound Juicer, en Windows puedes utilizar el programa CD extractor (CDex), ambos son software libre.

 

Extracción de Audio de un CD Audio con Sound Juicer

1. Arrancar el programa Sound Juicer.
Aplicaciones / Sonido y video / Extractor de CD Sound Juicer
2. Seleccionar las canciones a extraer.
3. Hacer clic sobre el botón Extraer

Por defecto Sound Juicer guarda los archivos generados en la carpeta_personal_usuario/artista/album i con el formato OGG.

Haciendo doble clic sobre el archivo generado se ejecuta automáticamente el programa para reproducirlo.
