Barra de herramientas de edición



Herramienta de selección. Permite seleccionar un fragmento de una pista y marcar el punto de inicio para la reproducción o la grabación.
Herramienta envolvente. Permite controlar el nivel de volumen de zonas seleccionadas.
Herramienta de dibujo. Permite modificar muestras individuales.
Zoom. Permite alejarse o acercarse para ver mejor la pista.
Herramienta de desplazamiento. Permite desplazar en el tiempo la pista.
Herramienta múltliple. Según la posición del cursor, se transforma en la herramienta de sección o en la envolvente automáticamente.
Las herramientas de cortar, copiar, pegar, deshacer, rehacer y zoom son las típicas de todos los programas.
Recorta fuera de la selección dejando sólo la parte selecconada.
Silencia la zona seleccionada
Ajusta la zona seleccionada en la ventana (en la ventana se muestra la zona seleccionada).
Ajusta el proyecto a la ventana (en la ventana se muestra todo el proyecto).
